# Udaipur Tour Travels - Travel Booking Website

A complete travel booking website with tour packages, car rentals, and booking management system.

## Features
- Browse tour packages and day trips
- User registration and login
- Book tours with date and people count
- Responsive design that works on all devices
- Contact form and inquiries
- Admin can manage tours

## Tech Stack
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose ODM
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Authentication**: JWT

## Installation

### Prerequisites
- Node.js installed
- MongoDB installed locally or MongoDB Atlas account
